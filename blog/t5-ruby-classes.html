Class example

12/19/14

My example class is meant to mimic a simple stopwatch with 3 buttons - start, stop, and reset. I will describe each method in the class and you can view the code here.

First, the initialize method. The Stopwatch class does not accept arguments. It has 3 instance variables - @accumulated, @start_time, and @status. @accumulated represents the total time from previously starting/stopping the stopwatch. @start_time is a timestamp of the last time the watch was started. @status can be either :running or :stopped depending on whether the watch is running.

The start method only does anything is the watch is current stopped. If it is stopped, then it will update @status to :running and record the current time in @start_time. This lets the watch keep track of how much time has passed since start was called.

The stop method only does anything if the watch is currently running. If it is running, then it will update @status to :stopped and add the different between the current time and @start_time to @accumulated. This way the watch will store previously accumulated time even if it is started again later.

The reset method sets all instance variables back to their initial state as though initialize had just been called.

Finally, the time method will return the current time that would be displayed on the face of a stopwatch. If the watch is stopped, then the time method will simply return @accumulated. If the watch is running then the time method will return the different between the current time and @start_time plus @accumulated.
